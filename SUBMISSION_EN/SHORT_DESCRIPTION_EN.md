# Microcontroller Systems - Short Description

This submission contains three completed laboratory works (Variant 5).

- **Lab 1**: 8-bit LED counter with two buttons (UP/DOWN). The counter is displayed in binary form on 8 LEDs.
- **Lab 2**: Clock configuration for target frequency (80 MHz for Variant 5), SysTick setup at 10 kHz, and output signal frequency verification.
- **Lab 3**: USART packet transmission on button press and USART receive control of LED state (`4` -> OFF, `5` -> ON).

Each lab folder includes:
- English report (`.docx` and `.md`),
- scheme/runtime screenshots,
- source code listing used for implementation.
