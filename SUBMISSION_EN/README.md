# Microcontroller Systems - Labs 1-3 (Variant 5)

This repository contains a complete submission package for three laboratory works.

## Content Overview

- `SHORT_DESCRIPTION_EN.md` - brief summary of all labs.
- `TASK_COMPLIANCE_EN.md` - checklist against the task requirements.
- `Lab1/` - materials for Laboratory Work 1.
- `Lab2/` - materials for Laboratory Work 2.
- `Lab3/` - materials for Laboratory Work 3.

## Folder Structure

### `Lab1/`
- `Lab1_Report_EN.docx` - final report (Word).
- `Lab1_Report_EN.pdf` - final report (PDF).
- `Lab1_Report_EN.md` - report source in Markdown.
- `Lab1_main.c` - firmware code listing.
- `Lab1_kicad_scheme.png` - schematic screenshot.
- `Lab1_wokwi_zero_state.png` - runtime screenshot (initial state).
- `Lab1_wokwi_run.png` - runtime screenshot (counter operation).

### `Lab2/`
- `Lab2_Report_EN.docx` - final report (Word).
- `Lab2_Report_EN.pdf` - final report (PDF).
- `Lab2_Report_EN.md` - report source in Markdown.
- `Lab2_main.c` - STM32 firmware code listing.
- `Wokwi_Lab2_sketch.ino` - Wokwi functional simulation sketch.
- `Lab2_wokwi_scheme.png` - schematic screenshot.
- `Lab2_wokwi_logic_analyzer.png` - measurement screenshot.
- `Lab2_wokwi_frequency_value.png` - frequency verification screenshot.

### `Lab3/`
- `Lab3_Report_EN.docx` - final report (Word).
- `Lab3_Report_EN.pdf` - final report (PDF).
- `Lab3_Report_EN.md` - report source in Markdown.
- `Lab3_main.c` - STM32 firmware with USART interrupt logic.
- `Wokwi_Lab3_sketch.ino` - Wokwi functional simulation sketch.
- `Lab3_wokwi_diagram.json` - Wokwi schematic connection file.
- `Lab3_wokwi_scheme.png` - schematic screenshot.
- `Lab3_wokwi_uart_tx.png` - terminal screenshot (TX packet sending).
- `Lab3_wokwi_uart_rx_led.png` - terminal screenshot (RX command and LED control).

- Lab 2 target frequency: **80 MHz**.
